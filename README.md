# SkillBridge 🎓

## Project Description
SkillBridge is a desktop-based freelancing platform for university students.
Freelancers can create gigs, clients can browse and order services, and both
can communicate, track jobs, and build reputation through ratings and reviews.
Built to simulate a real-world freelancing environment within the campus using
Object-Oriented Programming concepts in C++.
